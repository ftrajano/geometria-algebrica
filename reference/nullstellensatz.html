Nullstellensatz de Hilbert

"Teorema dos zeros" (Null = zero · stellen = lugares · satz = teorema). Hipótese sempre presente: k algebricamente fechado (ex.: ℂ).

Forma fraca

V(J) = ∅  ⟺  1 ∈ J  (⟺ J = k[x₁,…,xₙ]).

Leitura: um sistema de polinômios não tem solução comum ⟺ o 1 está no ideal gerado. Decidível por base de Gröbner (base = {1} ⟺ 1 ∈ J).

Forma forte

I(V(J)) = √J.

A volta J → V(J) → I(V(J)) devolve o radical de J, não J. A geometria esquece multiplicidade; o radical é tudo o que se perde.

Lembretes

SímboloSignificado
V(J)pontos onde todo polinômio de J se anula (variedade)
I(S)polinômios que se anulam em todo ponto de S (ideal)
√J{ a : aᵐ ∈ J p/ algum m ≥ 1 } (radical)
radicalideal com √J = J — exatamente os que vêm de variedades

Cuidado · precisa ser algebricamente fechado

Sobre ℝ: J = ⟨x²+1⟩ tem V(J) = ∅ mas 1 ∉ J → a forma fraca falha. Sobre ℂ: V(J) = {i,−i}, tudo coerente. Trabalhe em ℂ.

Contexto

Generaliza o Teorema Fundamental da Álgebra (1 polinômio em ℂ[x] tem raiz → sistemas em ℂ[x₁,…,xₙ] têm solução). É a ponte que torna a geometria calculável.
