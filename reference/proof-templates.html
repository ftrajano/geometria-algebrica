Anatomia de uma demonstração em álgebra

Modelos "Para mostrar…": como transformar uma definição em uma prova executável.

O segredo que quase ninguém ensina explicitamente: em álgebra abstrata, a definição do objeto é literalmente o roteiro da prova. Você quase nunca precisa de inspiração. Você precisa desempacotar a definição em uma lista de verificações e checar cada item. Este documento é essa lista, para os objetos que você mais vai encontrar.

Regra de ouro: ao ver "prove que X é um Y", escreva no papel a definição de Y e transforme cada cláusula em uma linha "Para mostrar…". A prova vira preencher lacunas.

O esqueleto universal

  1. Nomeie o objetivo. "Quero mostrar que S é um [ideal / subgrupo / homomorfismo]."
  2. Cite a definição de [ideal / …]. Cada cláusula vira uma obrigação.
  3. Introduza objetos genéricos. "Sejam a, b ∈ S e r ∈ R arbitrários." (Genérico = vale para todos.)
  4. Cheque cada cláusula, uma por vez, manipulando os objetos genéricos até cair de volta em S.
  5. Conclua. "Logo as três condições valem, então S é um ideal. ∎"

Modelo: provar que I é um ideal de um anel R

Definição → checklist

0 ∈ Io conjunto não é vazio / contém o zero
fecho (+)se a, b ∈ I então a + b ∈ I
absorçãose a ∈ I e r ∈ R então r·a ∈ I

Cuidado: a absorção é com qualquer r do anel inteiro — não só com elementos de I. É isso que torna ideal mais forte que subanel. Um ideal não precisa ser fechado sob a própria multiplicação como subanel; precisa absorver o anel todo.

Modelo: provar uma inclusão de conjuntos A ⊆ B

O movimento

  1. "Seja x ∈ A arbitrário. Quero mostrar x ∈ B."
  2. Desempacote o que "x ∈ A" significa (a definição de A).
  3. Manipule até obter a condição que define "x ∈ B".
  4. "Como x era arbitrário, A ⊆ B. ∎"

Para provar igualdade A = B: prove as duas inclusões, A ⊆ B e B ⊆ A, separadamente. Quando o objetivo é "todo elemento de A está em B", esse é sempre o esqueleto. Aninhe o movimento se a pertinência em B já for ela mesma um "para todo" (ex.: p ∈ V(I(S)) abre num "para todo polinômio f…").

Modelo: provar que H é um subgrupo de G

Definição → checklist (teste do subgrupo)

e ∈ Ho neutro de G está em H (logo não-vazio)
fecho (·)se a, b ∈ H então ab ∈ H
inversosse a ∈ H então a⁻¹ ∈ H

Atalho clássico (teste de um passo): H ≠ ∅ e a, b ∈ H ⇒ ab⁻¹ ∈ H.

Modelo: provar que φ é um homomorfismo de anéis

somaφ(a + b) = φ(a) + φ(b)
produtoφ(a·b) = φ(a)·φ(b)
unidadeφ(1) = 1 (na convenção com unidade)

Sempre começa do lado esquerdo e manipula até chegar no direito. Nunca os dois ao mesmo tempo.
