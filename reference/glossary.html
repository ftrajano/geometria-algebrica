Glossário

Os termos do nosso percurso, com a definição que usamos em todas as lições. Quando uma lição usa um símbolo, ele significa isto.

corpo · corpo algebricamente fechado · k[x₁,…,xₙ] · ideal · I(S) · V(F) · variedade afim · radical √J · ideal radical · Nullstellensatz

Corpo (field)

Conjunto com soma e produto onde dá para somar, subtrair, multiplicar e dividir (todo elemento não-nulo tem inverso). Exemplos: ℚ, ℝ, ℂ, corpos finitos 𝔽ₚ. É o "terreno" onde vivem os coeficientes dos polinômios.

Corpo algebricamente fechado

Um corpo k é algebricamente fechado se todo polinômio não-constante em k[x] tem pelo menos uma raiz em k.

Equivalências: (1) todo polinômio fatora em fatores lineares sobre k; (2) os únicos irredutíveis em k[x] têm grau 1; (3) nenhuma equação polinomial "vaza" para um corpo maior.

Sim: ℂ (Teorema Fundamental da Álgebra). Não: ℝ (x²+1 sem raiz real), ℚ (x²−2), corpos finitos.

Por que importa: é a hipótese de todo o Nullstellensatz. Sem ela o dicionário álgebra↔geometria vaza — sobre ℝ, ideais diferentes (⟨1⟩, ⟨1+x²⟩, …) dão a mesma variedade vazia. Trabalhe em ℂ. Fonte: CLO Cap. 4 §1, p. 176. Lição completa: 0005.

Anel de polinômios k[x₁,…,xₙ]

Todos os polinômios em n variáveis com coeficientes no corpo k. É um anel comutativo: soma e multiplica como esperado, mas em geral não há divisão. É o universo algébrico da geometria algébrica.

Ideal I ⊆ R

Subconjunto de um anel R que: (1) contém 0; (2) é fechado sob soma; (3) absorve produtos por qualquer elemento do anel (a∈I, r∈R ⇒ ra∈I). Ver Lição 0001.

Ideal de um conjunto I(S)

Dado um conjunto de pontos S ⊆ kⁿ: I(S) = { f ∈ k[x₁,…,xₙ] : f(p)=0 para todo p∈S } — os polinômios que se anulam em todo S. Sempre um ideal. Leva geometria → álgebra.

Variedade de um conjunto de polinômios V(F)

Dado F ⊆ k[x₁,…,xₙ]: V(F) = { p ∈ kⁿ : f(p)=0 para todo f∈F } — os pontos onde todos os polinômios se anulam. Leva álgebra → geometria. Ver Lições 0002–0003.

Variedade afim

Um conjunto de pontos da forma V(F) para algum conjunto de polinômios F — isto é, o conjunto-solução de um sistema polinomial. São os objetos geométricos centrais da área.

Radical de um ideal √J

√J = { a ∈ R : aᵐ ∈ J para algum m ≥ 1 } — os elementos cuja alguma potência cai em J. É um ideal, e contém J. "Apaga expoentes": √⟨x²⟩ = ⟨x⟩. Ver Lição 0004.

Ideal radical

Ideal que é igual ao próprio radical: √J = J. Pelo Nullstellensatz, são exatamente os ideais que vêm de variedades (I(V(J)) = J).

Nullstellensatz

Fraco: sobre k algebricamente fechado, V(J)=∅ ⟺ 1∈J. Forte: I(V(J)) = √J. Detalhes no cartão do Nullstellensatz e na Lição 0003.
